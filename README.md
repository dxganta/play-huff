## Contracts

1. [SafeMath.huff](https://github.com/realdiganta/play-huff/blob/main/contracts/SafeMath.huff)

## Setup

```sh
npm i
```

### Building and Testing

To compile your contracts, write:

```sh
npx hardhat compile
```

To test your contracts, write:

```sh
npx hardhat test
```
